OOPs Concepts - Interview Prep

1. Encapsulation

Binding data and methods. Achieved using classes and access modifiers. Improves security and modularity.

2. Inheritance

Code reusability by inheriting base class properties. Supports hierarchical class design.

3. Polymorphism

Same interface, different behavior. Includes method overloading (compile-time) and overriding (runtime).

4. Abstraction

Hiding internal details. Achieved via abstract classes and interfaces. Reduces complexity.
