Coding Interview - Must Know

1. Arrays & Strings

Sliding window, 2 pointers, prefix sums, hash maps.

2. Binary Search

Log(N) search in sorted arrays. Variations: first/last occurrence, peaks, rotated arrays.

3. Recursion & Backtracking

Used in N-Queens, permutations, combinations. Watch stack overflow cases.

4. Dynamic Programming

Overlapping subproblems. Key: states, transitions, memoization/tabulation.
